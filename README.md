# RExI Study Intake

Interactive prototype for the RExI Study Intake form.

Live site: `https://<your-github-username>.github.io/rexi-study-intake/`

## GitHub Pages Setup

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch**, branch `main`, folder `/` (root)
4. Save — the site will be live in ~60 seconds
